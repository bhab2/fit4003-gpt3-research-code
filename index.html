Prompt Engineering and GPT-3


Description of the Taxonomy

We have attempted to classify GPT-3 applications into five primary categories namely Developer Productivity, Image Generation, Media Content Creation, Text Conversion, and Conversational.â€¨ Developer Productivity category covers the technical tasks accomplished by GPT-3 applications which may not be used by the general public or users. The Images category includes applications which can generate any type of imagery based on natural language inputs or accomplish any other tasks centered around imagery. Multimedia covers applications that deal with natural language and various other forms of media, such as graphic design, videos, music, and others. Text Conversion is quite a broad category which has sub-categories such as Text Compression, Text Expansion, and Text Translation. These mainly include tasks which involve working with pure text inputs and outputs. Conversational category groups all applications which have capabilities to communicate or talk with another human like a human. They can be further divided into general conversations (e.g. Personal Companions) or specific conversations about certain topics (e.g. Health Support) or conversations to get tasks done (Business Assistants).
